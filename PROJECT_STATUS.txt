âš ï¸  IMPORTANT NOTES
================================================================================

1. Data Persistence:
   - All contacts stored in browser's localStorage
   - Data persists across browser sessions
   - Survives browser restart but NOT cache clear
   - Each browser/device has separate data

2. Security:
   - Data stored locally (not sent to servers)
   - No authentication required
   - No external API calls

3. Limitations:
   - Storage limited by browser (usually 5-10MB)
   - Not synced across devices
   - Clearing cache will delete all contacts

4. Browser Storage:
   - localStorage key: 'contacts'
   - Max ~100,000 contacts (depending on field content)
   - No backup/restore built-in
